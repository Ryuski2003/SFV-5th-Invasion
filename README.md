# SFV-5th-Invasion

## How to Install
You will need FMM: Fluffy's Mod Manager

1. put mod folders in "FMM DIRECTORY" \Games\SF5\Mods
2. Launch Mod manager
3. Click Define game
4. Select SF5
5. click manage mods
6. click read game archives
7. enable mods in manage mods tab
8. play
